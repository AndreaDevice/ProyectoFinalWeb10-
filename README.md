# ProyectoFinalWeb10
Proyecto final para  Aplicaciones Web Progresivas - 1002

------------------------------------------------------------------------------------------------------------

Proyecto:
1.- Insertar.

2.- Interfaz para ingresar datos:

  2.1.- Usuario
  
  2.2.- Password
  
  2.3.- Nombre
  
  2.4.- Perfil
  
3.- Mensaje en pantalla "Se inserto correctamente" o "Hubo un error, con link para ingresar "Ver registros".

4.- Interfaz de la tabla donde se ven los datos. Con acciones "Alta de usuario" y "Eliminar".

  4.1.- Interfaz de eliminacion con el mensaje "El usaurio identificado como xxxx sera eliminado, con un link para regresar a la pagina de datos.

------------------------------------------------------------------------------------------------------------

Base de datos, tablas:

  1.-  perfil
    -id: (ejemplo 1, 2, 3)
    -nombre: por ejemplo ("Administrador", "Empleado", etc.)

  2.- usuario
    usuario: el nombre de usuario (login), máximo 10 caracteres.
    pass: la contraseña, máximo 10 caracteres.
    nombre: nombre completo del usuario.
    idperfil: a qué perfil pertenece (1 o 2 según la tabla perfil).

------------------------------------------------------------------------------------------------------------

Criterios:

  1.- Inserción de registros en la Base | 20 Pts.
  
  2.- Consulta de registros de la Base de Datos | 20 Pts.
  
  3.- Baja de registros de la Base de Datos “Eliminar” | 20 Pts.
  
  4.- Actualización de registros de la Base de Datos | 25 Pts.
  
  5.- Adecuaciones y adaptaciones de Diseño/Interfaz del Proyecto (Revise Archivo de Hojas de Estilo Adjunto y complemente) | 10 Pts.
  
  6.- Entrega en tiempo y forma: 29 de Noviembre del 2025 | 5 Pts.
